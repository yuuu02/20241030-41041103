Chart.js 動態圖表套件

Chart.js 是一個簡單且靈活的 JavaScript 圖表繪製函式庫，可以幫助開發者在網頁上創建各種互動式圖表。

Chart.js 官網

套件介紹

Chart.js 提供了豐富的圖表效果，包括：

  • 折線圖 (Line Chart)
  • 柱狀圖 (Bar Chart)
  • 圓餅圖 (Pie Chart)
  • 散點圖 (Scatter Chart)
  • 雷達圖 (Radar Chart)
  • 區域圖 (Area Chart)

基本使用方法

    // 1. 引入 Chart.js
    <script src="https://cdnjs.cloudflare.com/ajax/libs/Chart.js/4.4.0/chart.umd.min.js"></script>

    // 2. 準備 Canvas 元素
    <canvas id="myChart"></canvas>

    // 3. 初始化圖表
    const ctx = document.getElementById('myChart').getContext('2d');
    const myChart = new Chart(ctx, {
        type: 'bar',
        data: {
            labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'],
            datasets: [{
                label: '投票數',
                data: [12, 19, 3, 5, 2, 3],
                backgroundColor: [
                    'rgba(255, 99, 132, 0.2)',
                    'rgba(54, 162, 235, 0.2)',
                    'rgba(255, 206, 86, 0.2)',
                    'rgba(75, 192, 192, 0.2)',
                    'rgba(153, 102, 255, 0.2)',
                    'rgba(255, 159, 64, 0.2)'
                ],
                borderColor: [
                    'rgba(255, 99, 132, 1)',
                    'rgba(54, 162, 235, 1)',
                    'rgba(255, 206, 86, 1)',
                    'rgba(75, 192, 192, 1)',
                    'rgba(153, 102, 255, 1)',
                    'rgba(255, 159, 64, 1)'
                ],
                borderWidth: 1
            }]
        },
        options: {
            scales: {
                y: {
                    beginAtZero: true
                }
            }
        }
    });
