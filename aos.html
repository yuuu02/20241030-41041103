AOS (Animate On Scroll) 動畫套件

AOS 是一個能讓元素在捲動時產生動畫效果的 JavaScript 套件。

AOS官網

套件介紹

AOS 提供了豐富的動畫效果，包括：

  • 淡入淡出效果
  • 滑動效果
  • 翻轉效果
  • 縮放效果

基本使用方法

// 1. 引入 CSS 和 JS
<link href="https://cdn.jsdelivr.net/npm/aos@2.3.4/dist/aos.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/aos@2.3.4/dist/aos.js"></script>

// 2. 初始化 AOS
<script>
    AOS.init();
</script>

// 3. 在HTML元素上使用
<div data-aos="fade-up">內容</div>

進階設定

AOS.init({
    duration: 1000,    // 動畫持續時間
    delay: 100,        // 延遲時間
    once: true,        // 動畫是否只執行一次
    offset: 120,       // 觸發偏移
    easing: 'ease-in-out-back'  // 緩動函數
});
