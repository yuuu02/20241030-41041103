套件說明

Lottie 是由 Airbnb 開發的開源動畫庫，能夠在網頁、iOS、Android 和 Windows 上渲染 Adobe After Effects 動畫。具有以下特點：

  • 可直接使用 After Effects 製作的動畫，支援多種動畫效果
  • 高效能的向量動畫渲染，檔案小且清晰度高
  • 跨平台支援 (Web、iOS、Android、Windows)
  • 支援程式控制播放、暫停、速度等動畫參數
  • 豐富的動畫資源庫，可直接使用現成動畫

使用方式

// 1. 安裝套件
    npm install lottie-web

    // 2. 初始化動畫
    const animation = lottie.loadAnimation({
        container: document.getElementById('animation-container'),
        renderer: 'svg',
        loop: true,
        autoplay: true,
        path: 'path/to/animation.json'
    });

    // 3. 控制動畫
    animation.play();     // 播放
    animation.pause();    // 暫停
    animation.stop();     // 停止
    animation.setSpeed(2); // 設定速度
