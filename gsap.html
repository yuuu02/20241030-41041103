GSAP 動畫效果

GSAP官網

套件說明

GSAP (GreenSock Animation Platform) 是一個專業級的 JavaScript 動畫庫，具有以下特點：

  • 高效能的動畫引擎
  • 豐富的動畫效果和控制選項
  • 支援 CSS 屬性、SVG、Canvas 等多種動畫對象
  • 時間軸功能，可輕鬆創建複雜的動畫序列
  • 提供豐富的緩動函數

範例程式碼





gsap.to(".box1", {
    duration: 2,
    x: 200,
    rotation: 360,
    scale: 1.5,
    ease: "power2.inOut"
});


const tl = gsap.timeline({repeat: -1});
tl.to(".box2", {duration: 1, x: 200})
  .to(".box2", {duration: 1, y: 200})
  .to(".box2", {duration: 1, x: 0})
  .to(".box2", {duration: 1, y: 0});


gsap.to(".stagger-box", {
    duration: 1,
    y: 100,
    opacity: 0,
    stagger: 0.2,
    ease: "back.in"
});
